字节跳动张羽乌镇大会发言：互联网为公益创新提供了新思路

10月21日，在第六届世界互联网大会“互联网公益慈善论坛”上，字节跳动副总裁张羽分享了互联网公益的三个关键词：赋能、创新、人人公益。张羽表示：互联网赋能公益，公益让互联网变得更暖。在世界互联网大会上设置公益主题的论坛，真正体现了互联网普惠的特性。“随着数字技术的持续发展，互联网打破了传统壁垒，为社会各界参与公益创新提供了新的通道和新的思路。创新者正在利用互联网改变行业、改变社会。”

“只有用互联网技术赋能公益，才能真正让公益项目高效率、大规模、可持续。”张羽以自“头条寻人”项目举例：头条寻人是一个很典型的互联网赋能公益的项目：它借助“互联网+”的精准地域弹窗技术，对寻人或寻亲信息进行定向地域推送，来帮助家属寻找走失人员，也帮助在救助管理机构的疑似走失人员寻找家人。

“60秒，是该项目找回一位走失者最短的时间。截止到2019年10月，项目已成功找回的走失者11406人。在互联网时代之前，用秒来度量一次成功的寻人信息传递，这恐怕是不能想象的。”张羽说。
